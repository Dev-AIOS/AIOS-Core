# 🧠 AI Reseller — المنصة العربية الموحدة للذكاء الاصطناعي

## المستند الاستشاري الكامل (2025–2030)

**مشروع:** منصة ذكاء اصطناعي عربية شاملة للمطورين والشركات — المعادل العربي لـ ChatGPT + OpenRouter + Cursor + Claude + Gemini في نظام بيئي واحد.

**المؤسس:** فردي | **الميزانية:** شبه صفرية | **السوق:** مصر ← الشرق الأوسط ← العالم  
**نموذج العمل:** تجميع وإعادة بيع نماذج AI (Aggregator/Reseller) + اشتراكات بالجنيه المصري  
**الميزة التأسيسية:** فوترة بالجنيه المصري عبر بوابات محلية (Paymob, Fawry, Vodafone Cash, InstaPay) — بلا حاجة لبطاقة دولية

## الملخص التنفيذي

### الفكرة
منصة عربية تجمع كل قدرات الذكاء الاصطناعي في مكان واحد — محادثة، نماذج متعددة، برمجة، صور، تحليل مستندات، وكلاء، أدوات مطورين، API — كل ذلك بفوترة **بالجنيه المصري** عبر وسائل الدفع المحلية. لا حاجة لبطاقة دولية ولا اشتراك بالدولار.

### المشكلة
المستخدم المصري والعربي يعاني من:
- حاجته لبطاقة ائتمان دولية للاشتراك في ChatGPT وخدمات AI الأخرى
- سعر ChatGPT Plus ~1000 ج.م شهرياً (بسعر $20)
- لا توجد منصة عربية موحدة تجمع كل النماذج والأدوات
- لا توجد خيارات دفع محلية (Fawry, Vodafone Cash, InstaPay, Paymob)
- لا توجد خطط تقسيط (valU, سول, أمان)

### الحل
منصة واحدة تقدم كل شيء بالجنيه وبواجهة عربية وبأسعار تبدأ من **75-150 ج.م شهرياً** فقط.

### نموذج العمل
- **التجميع (Aggregation):** نشتري حق الوصول لنماذج AI بالجملة (عبر OpenRouter ومزودين مباشرين) ونعيد بيعه بهامش 25-60%
- **الاشتراكات:** خطط شهرية/سنوية تناسب كل فئات الدخل المصري
- **API للمطورين:** وصول مرن بنظام رصيد مسبق الدفع
- **Marketplaces:** عمولات من بيع البرومبتات والوكلاء وخوادم MCP
- **Enterprise و White-label:** عقود B2B وحكومية

### الجدوى المالية — أرقام محورية

## تحذيرات مهمة ⚠️
1. **كل الأسعار** في هذا المستند هي تقديرات كما في منتصف 2025. أسعار النماذج تتغير أسبوعياً — تحقق دائماً من openrouter.ai/models و artificialanalysis.ai
2. **سعر الصرف** 1$ = 50 ج.م هو افتراض متغير — يجب إعادة حساب السيناريوهات المالية بانتظام
3. **هذا ليس استشارة قانونية** — الجزء 10 يحتوي على قائمة مراجعة قانونية يجب مراجعتها مع محامٍ ومحاسب مصري قبل الإطلاق
4. **المستند مبني على المعرفة العامة** — بعض التفاصيل الدقيقة قد تحتاج تحديثاً عند التطبيق الفعلي
